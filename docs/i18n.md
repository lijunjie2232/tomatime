# 国际化 (i18n) 支持

本项目现已支持国际化，可以自动根据用户的系统语言显示相应的界面语言。

## 支持的语言

目前支持以下语言：
- 简体中文 (zh)
- 英语 (en)
- 日语 (ja)
- 西班牙语 (es)
- 俄语 (ru)

## 如何添加新的语言支持

1. 在 `app/src/main/res/` 目录下创建新的语言文件夹，命名格式为 `values-[语言代码]`
2. 在新文件夹中创建 [strings.xml](file:///home/li/data/code/android/exploit/tomatime/app/src/main/res/values/strings.xml) 文件
3. 参考现有语言文件，翻译所有字符串资源
4. 确保所有字符串名称保持一致

示例：
```xml
<?xml version="1.0" encoding="utf-8"?>
<resources>
    <string name="app_name">应用名称</string>
    <!-- 添加其他字符串 -->
</resources>
```

## 测试不同的语言

要测试应用在不同语言下的显示效果：

1. 在设备或模拟器上打开设置
2. 进入语言和输入法设置
3. 更改系统语言
4. 重新启动应用查看效果

## 字符串使用规范

在代码中应始终使用字符串资源，而不是硬编码字符串。这样可以确保应用能够正确地进行本地化。

示例：
```kotlin
// 正确用法
text = getString(R.string.app_name)

// 错误用法
text = "应用名称"
```
